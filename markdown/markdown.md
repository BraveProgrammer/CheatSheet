# **Markdown Cheat Sheet**

**Requirements:**  
&nbsp;&nbsp;&nbsp;&nbsp;1. Nothing!

# Heading

```markdown
# head level 1
## head level 2
### head level 3
#### head level 4
##### head level 5
###### head level 6
```

# Bold

```markdown
this is a **bold** text
and this __bold__
```

# italic

```markdown
this is a *italic* text
and this _italic_
```

# Blockquote

```markdown
> blockquote
> and this
```

# Ordered List

1. First Item
2. Second Item

# Unordered List

+ First Item
+ Second Item

# Code

```markdown
‎```language
your code
‎```
```

or

```markdown
`code`
```

# Horizontal Rule

```markdown
---
```

# Link

```markdown
[title](https://www.example.com)
```

# Image

```markdown
![alt text](image.jpg)
```

# Table

```markdown
| Syntax    | Description |
| --------- | ----------- |
| Header    | Title       |
| Paragraph | Text        |
```

# Footnote

```markdown
Here's a sentence with a footnote. [^1]

[^1]: This is the footnote.
```

# Heading ID

```markdown
### My Great Heading {#custom-id}
```

# Definition List

```markdown
term
: definition
```

# Strikethrough

```markdown
~~The world is flat.~~
```

# Task List

```markdown
- [x] Buy new phone
- [ ] Update the pc
- [ ] Update the website
```
